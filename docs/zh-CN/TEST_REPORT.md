## 测试结果摘要

全部 85 条用例通过。覆盖范围包括：

- **服务端 / Node（tests/mod.test.ts，71
  条）**：哈希（SHA1、SHA256、SHA512）、哈希验证、随机数生成（字节、字符串、整数、UUID）、AES
  密钥生成与导入、对称加密/解密（AES-GCM、AES-CBC）、RSA/ECDSA 密钥对生成、RSA
  加密/解密、数字签名（RSA、ECDSA）、JWT（签发、验证、解码）。
- **浏览器客户端（tests/browser.test.ts，14 条）**：在真实浏览器中通过
  @dreamer/test 浏览器集成（无头）执行相同 API，包括
  hash、verifyHash、generateUUID、AES 加解密、JWT
  签发/验证/解码、generateRandomBytes/String/Int、decodeJWT、hashPassword/verifyPassword（预期需外部库）、Uint8Array
  密钥加解密（importAESKey）、RSA 加解密往返、RSA 签名验证往返。
